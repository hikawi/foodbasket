# FoodBasket - Backlog

## 1. POS System Epic â€“ Staff Tools (Tablet-first)

Goal: Provide smooth, role-aware tools for waiters, chefs, managers in a
tenant/branch context.
Device priority: Tablet (>=10", landscape, touch), fallback to desktop/phone.

Prioritized Stories:

1. Staff login & tenant auto-scope
2. View active / new orders dashboard
3. Accept / prepare / complete an order
4. Menu Editor: View + Add / Edit / Disable item
5. Users Editor: View staff list + Add / Edit / Deactivate user (manager only)
6. Basic table / seat management
7. Order notes / modifiers / voids
8. Logout / session timeout

Out of scope for initial POC: Full KDS (kitchen display), inventory sync,
reporting, offline mode, advanced scheduling.
