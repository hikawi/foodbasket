# ES00: Tech Stack

## Summary

This documents aims to provide a quick insight into what technology stack will be
used for the software and the architecture of the platform.

## Technology Stacks

- Frontend Technologies:
  - Vue SPA (Progressive Web Application architecture)
  - Astro and Vue (Content-based landing pages)
  - TailwindCSS (CSS Frameworks for quickly prototyping UIs)
- Backend Technologies:
  - Golang (Backend language)
  - Echo (Go's HTTP server framework)
- Infrastructure Technologies:
  - Valkey or ElastiCache, for caching sessions and frequently accessed data.
  - A SMTP server, for sending emails, such as promotional emails.
